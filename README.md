# Color Palette Generator 🎨

Color Palette Generator is a lightweight web tool that creates random color palettes for designers and developers.

With a single click, the tool generates five random colors along with their hex codes. Users can quickly copy any color value and use it in their designs or projects.

The goal of this tool is to make exploring color combinations fast and simple.

Live Website:
https://color-jagratahuja.vercel.app/


---

## ✨ Features

* Generate random color palettes
* Each palette contains 5 colors
* Display hex color codes
* Copy hex code to clipboard
* Clean minimal interface
* Responsive layout

## ⚙️ How It Works

1. Click **Generate Palette**.
2. The system generates five random colors.
3. Each color is displayed in a card with its hex value.
4. Click **Copy** to copy the hex code to your clipboard.
